## Purpose
The Auto Writer generates high-quality written content automatically. It can create emails, articles, captions, social posts, formal texts, and casual messages.

## Outputs
- Full polished text
- Multiple variations (formal, friendly, creative)
- Human-like version and more formal version

## Inputs
- Topic
- Text type
- Tone
- Desired length

## No-Code MVP Version
ChatGPT acts as the main text generator. The user only provides the initial settings (topic, tone, type).

## User Flow
1. User provides topic or instructions
2. User selects tone and text type
3. System generates the final text and variations
